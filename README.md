# 🌐 World of Vibecraft

**AI-Assisted Development Studio** — разработка со скоростью мысли. Собираем MVP за 5 дней с помощью ИИ-инженеров.

---

## 🚀 О проектe

World of Vibecraft — лендинг студии ИИ-разработки. Мы не пишем код построчно — мы оркестрируем агентов, фреймворки и LLM-пайплайны, чтобы вы получили работающий продукт за дни, а не месяцы.

### Что делаем:
- **MVP за неделю** — full-stack веб-приложение, база данных, авторизация
- **Из Figma в Продакшн** — точный перенос дизайна в чистый код
- **AI-Автоматизация под ключ** — LLM-воркфлоу, парсеры, умные чат-боты

---

## 🛠 Стек

- **Frontend:** Next.js, TailwindCSS
- **Backend:** Supabase, PostgreSQL
- **Deploy:** Vercel
- **AI:** ИИ-агенты, LLM-пайплайны

---

## ✨ Особенности лендинга

- 🎨 **Необрутализм дизайн** — жирные бордеры, яркие акценты, offset-shadow
- 🌙 **Тёмная/светлая тема** с сохранением в localStorage
- 🎬 **Анимации:** glitch, neon pulse, shake, marquee, floating deco
- 📱 **Адаптивная вёрстка** (mobile-first)
- 🔒 **Валидация форм** — защита от XSS, инъекций, с реальной санитизацией
- ⚡ **Zero dependencies** — чистый HTML/CSS/JS, без CDN, без сборки

## 📦 Локальный запуск

Просто откройте `index.html` в браузере:

```bash
# Открыть напрямую
open index.html

# Или через простой HTTP-сервер
python -m http.server 8000
# → http://localhost:8000
```
